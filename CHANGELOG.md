## [1.0.0] — 2026-03-09

### Added
- CDE/Motif desktop environment in a single `index.html` file
- Front Panel: draggable, collapsible, with analog clock, HDD LED, workspace switcher
- Window manager: draggable, resizable Motif windows with orange focused titlebar
- Terminal emulator with virtual Debian-like filesystem and 30+ commands
- File Manager with icon/list view, breadcrumb navigation, virtual `/` hierarchy
- Text Editor (Notepad) with open/save on the virtual filesystem
- Chess game with minimax AI opponent (depth 2, alpha-beta pruning)
- Style Manager: desktop colour, titlebar gradient, terminal font
- Simulated Browser, FTP Client, IRC Client, Application Manager
- Desktop icons (Terminal, resume.txt)
- Right-click context menus on windows and desktop
- `CONFIG` block for personalising username, hostname, and MOTD
- Browser compatibility: Chrome 60+, Firefox 55+, Safari 11+, Edge 79+
- GitHub Pages ready — no build step required
